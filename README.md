# Solia Documentation

## Stack

- Next.js
- Vercel

## Local Development

```bash
npm install
npm run dev
```

## Deployment

Deploy on Vercel and connect the custom domain:

1. Import `solia-docs` repository
2. Open project `Domains`
3. Add `docs.solia.network`
